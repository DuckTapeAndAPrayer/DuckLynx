# DuckLynx
This project is a suite of smaller projects to allow the replacement of firmware on and repair of REV Robotics Expansion Hub and Control Hub devices. This projects refers to the hubs by their codename: 'Lynx'. This project also focuses specifically on the Hardware Interface Board (HIB). This board is present in both the Expansion and Control Hubs and is what teams plug their devices into. The Control Hub has an additional Android board to allow it to work without a separate RC.

## Subprojects
### Firmware
This is a replacement firmware for the Lynx HIB. It is NOT legal for competition use for FTC. DO NOT use it in competition. It currently supports a subset of the features of the stock firmware.

### Info
This folder contains info and documentation about the Lynx and Stock Firmware.

### DuckUpdate
This folder contains a utility to flash the firmware of Lynx HIBs
